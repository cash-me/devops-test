# Teste para vaga de DevOps
Teste da CashMe para a vaga de DevOps

## Introdução

O objetivo do teste é avaliar a experiência de DevOps e a capacidade do candidato de resolver problemas. 
Sabemos que o prazo é um desafio e que atingir a perfeição será impossível, então a priorização dos esforços é importante e também será avaliada.

## Descrição da tarefa

1. Criar 2 contas que utilizem o free-tier da AWS, uma para QA e outra para PROD
2. Fazer um fork da aplicação https://expressjs.com/pt-br/starter/hello-world.html no GitHub
3. Configurar o kubernetes nas contas da AWS usando terraform
4. Criar pipeline de deployment da aplicação para os 2 ambientes. Usar ferramenta de CI que preferir (ex.: CircleCI) 
5. Criar endpoints públicos das aplicações de QA e PROD com HTTPS

## Observações

1. A avalição será feita 100% no que estiver versionado no Github. Não serão aceitos códigos ou planos de teste enviados por outro canal (ex.: e-mail)
2. O prazo para realizar o teste é de uma semana já considerando que o candidato trabalhe no horário comercial
3. Qualquer dúvida ou dificuldade entre em contato com o recrutador ou com o responsável técnico pelo teste
